Build a Storybook-based component library in my existing Next.js e-commerce project.

Requirements:

* Install and configure Storybook.
* Create stories for 3 reusable components:

  * Button
  * Input Field
  * Product Card
* Add multiple variants using Storybook Args and Controls.
* Button variants: Primary, Secondary, Disabled.
* Input variants: Default, Error, Disabled.
* Product Card variants: Default, Discounted, Out of Stock.
* Enable Light Mode and Dark Mode theme switching.
* Ensure all components are fully isolated and viewable inside Storybook.
* Configure and verify `npm run build-storybook`.
* Prepare the project for deployment on Vercel or Chromatic.
* Follow clean folder structure, reusable component design, and Storybook best practices.
